# Privacy Guardian  
Smart Sensor Monitor v2.0  

An Android application that analyzes app permissions and detects potential privacy risks based on sensor access (Camera, Microphone, Location, etc.).

---

##  Features

-  Real-time Privacy Risk Score for apps  
-  Detailed app-level analysis  
-  Permission audit (Camera, Mic, GPS, Contacts, Storage)  
-  Risk classification (Low, Medium, High, Critical)  
-  Sensor exposure tracking (7-day trends)  
-  Actionable recommendations for users  

---

##  Problem it Solves

Most users don’t understand how much access apps have to sensitive data.

Privacy Guardian helps users:
- Identify risky apps  
- Understand permissions clearly  
- Take action to protect their data  

---

##  Tech Stack

- Java  
- Android Studio  
- Android SDK  
- UI/UX Design (Custom Dark Theme)
